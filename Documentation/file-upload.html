Uploaded file size configuration

To change the limit of maximum size of uploaded file, and file type please follow these steps

Image size

  1. open backend > app > Models > Helper > Validation.php file.
  2. Find the imageRules function
  3. Change your desired file type and maximum image file size.

Find the videoRules function and change your desired file type and maximum video file size.

Video size

  1. open backend > app > Models > Helper > Validation.php file.
  2. Find the videoRules function
  3. Change your desired file type and maximum image file size.

For cPanel

To change the limit of maximum size of uploaded file please follow the steps below

  1. Log into your cPanel account
  2. Go to the File Manager
  3. Navigate to the folder where the .htaccess file is located(Admin panel code)
  4. Right-click on the .htaccess file and select "Edit"
  5. Add these following line to the file:

    6. php_value memory_limit 10M

    php_value post_max_size 30M

    php_value upload_max_filesize 10M
  6. Save the changes and close the file
  7. Refresh the website to see the changes take effect

Note: The .htaccess file may not be present in the folder by default. In that case, you will need to create a new file named .htaccess and add the above code.

For VPS server

To change the limit of maximum size of uploaded file, please follow the steps below

  1. SSH to your server
  2. Run nano backend > php > local.ini
  3. Change the value of upload_max_filesize and post_max_size.
  4. Restart the server. Depends on your server, how to do so. If you are using docker, please stop the docker container by running the docker-compose down in the root of the project. Then run docker-compose up -d to start the container.
