Folder structure

Unzip the package file.

You see the following folders:

  • Install.zip
  • Updates
  • Docker
  • Docker_Update
  • Documentation

Install.zip

This folder is to install the project in the server.

Updates

To update the a new version this files will be required.

Docker

This folder contains the required source code to install the script with docker. It's optional. A VPS server server is required.

Docker Update

This folder contains the required source code to update the script with docker.

Documentation

This folder contains the documentation.
