Login with Facebook

Create Facebook App

We need facebook client id and secret that way we can get information of other user.

  1. Please go to Facebook Developers Console. You can find bellow screen:
  2. Now you have to click on Create app.
  3. Choose a type of the app.
  4. Put details of the app.
  5. Now click Setting > Basic and set the App domains.
  6. Now scroll to the bottom of the page and click Add platform. Click Website from the popup.
  7. Put the URL of your server and save.
  8. Now click Add product. Then set up the Facebook login.
  9. Click Web.
  10. Put the URL of your server.
  11. Click the Facebook Login > Setting from left sidebar.
  12. Put your admin panel's redirect URL in Valid OAuth Redirect URIs. eg. https://admin.yourdomain.com/api/v1/user/social-login/callback/facebook

Setting the Google configuration

Now it's time to put the client id and secret in the server.

  1. Log in to your admin panel.
  2. Go to Settings > Social Login
  3. Change the Facebook Client Id and Facebook Client Secret.

Clearing cache

Now it's time to clear the cache of the backend.

  1. Open admin.yourdomain.com/api/clear-cache in the browser. Cache has been cleared.

You can also clear the cache from the admin panel also.

  1. Login to your Admin panel.
  2. Go to Setting > Clear cache.
  3. And click Clear cache button. Cache has been cleared.
